Информация

Консольные переменные

cr_hud 1/0

Позволяет включить/отключить HUD сервера.

cr_hud_entinfo 1/0

Позволяет включить/отключить информацию об энтити.

0 - выключено

1 - включено (только в Build)

2 - включено всегда

cr_tts_voice <голос>

Устанавливает голос Text-To-Speech.

Список доступных голосов:

  • oksana
  • zahar
  • ermil
  • jane
  • omazh
  • google

Команды

Все команды начинаются с “!” в чате. Чтобы посмотреть все доступные команды, можете зайти в меню (!menu).

build, билд, игшдв, строить/pvp, rdm, пвп, змз, убивать, резня/rp, рп, кз

Переход в режим строительства/PvP/РП.

dropmoney <кол-во>

Выкинуть деньги.

ranksync

Синхронизировать ранг сервера Garry’s Mod с сервером Discord.

switchtp

Переключить разрешение телепортации к себе.

switchdamage <игрок>

Переключить разрешение нанесения урона по себе от игрока.

name, setname, nick, setnick <имя>

Изменить никнейм в игре.

resetname

Сбросить изменённый никнейм.

Expression 2

string entity:crPlyGetGamemode

Возвращает режим игрока.

string entity:crPlyGetRank

Возвращает ранг игрока.

number entity:crPlyIsRagdolled

Возвращает 1, если игрок превращён в рэгдолл.

number entity:crPlyIsHandcuffed

Возвращает 1, если игрок в наручниках.

entity entity:crDoorGetOwner

Возвращает владельца двери.
