FAQ

Почему нельзя использовать обычный дубликатор?

Обычный дубликатор запрещён, вместо него используйте Advanced Duplicator 2.

Это сделано во избежание краша сервера.

Почему не работает Media Player?

Вам нужно установить версию Garry’s Mod - Chromium

  1. Зайдите в библиотеку игр в Steam, найдите Garry’s Mod
  2. Нажмите по нему ПКМ → Свойства → Бета-версии и выберите из списка x86-64 - Chromium + 64-bit binaries
  3. Дождитесь завершения установки и заходите на сервер

Почему друг не может брать мои пропы?

На сервере установлена защита пропов. Игроки не могут брать чужие пропы. Но вы можете добавить друга в Buddies

  1. Зайдите в Спавн-Меню (Q)
  2. В разделе Утилиты найдите Falco’s Prop Protection, в котором будет подраздел Buddies
  3. Нажмите на никнейм друга в списке
  4. Выберите, к чему он будет иметь доступ

Как отключить третье лицо?

Достаточно поменять консольную переменную thirdperson_etp на 0

Настройки третьего лица также можно найти в Спавн-Меню → Утилиты → Пользователь - Thirdperson

Также существует переменная thirdperson_etp_vehicles_sync, которая отвечает за синхронизацию третье лица с транспортом.
Хотя по умолчанию эта функция отключена, чаще всего именно из-за неё начинается путаница, поэтому следует проверить, отключена ли она

Как получить роль ранга в дискорде?

  1. Зайти на сервер
  2. В Discord выполнить команду /link
    Свой SteamID можно скопировать из расширенной панели игрока (ЛКМ по строке игрока) в TAB
  3. Принять запрос на сервере

Эти простые действия достаточно выполнить всего один раз. При получении нового ранга, он автоматически синхронизируется с Discord.
Если что-то вдруг пошло не так, можно выполнить команду !sync rank на сервере

Как открыть меню Звуковых Насмешек?

Меню открывается с помощью консольной команды +cr_soundtaunts
Для удобства её можно забиндить на любую клавишу: bind t +cr_soundtaunts
